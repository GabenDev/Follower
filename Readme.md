- Node and npm
⋅⋅⋅ curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -

..* sudo npm install -g cordova

- Ionic
..* sudo npm install -g ionic

- MongoDB
..* sudo apt install mongodb-server
..* sudo mkdir /data/db
..* sudo chown -R `whoami` /data
